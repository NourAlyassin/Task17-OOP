# Task17-OOP
First OOP assignment, implementing a Product class with properties and methods.

Requirements:

Define a Product class.
Properties: name, price, brand, image, description, tax.
Methods: getName(), priceAfterDiscount(number), getFinalPrice().
Display all data in a Bootstrap card.

Steps:
Define the class and create three objects with all property values.
Use the methods on each object.
Display the final data in Bootstrap cards.
